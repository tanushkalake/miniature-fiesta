Goals

  1. Understand and use the vocabulary of web design and development industry.
  2. Create websites with graphics, navigation, and advanced interactivity.
  3. Read, write and implement creative project briefs.
  4. Produce site architecture and page wireframes for UIX design.
  5. Understand best practices of fonts and images use for web development across multiple platforms and devices.
  6. Create web pages that demonstrate an understanding of the elements and principles of graphic design and usability with layout, proximity, balance and appropriate color usage.
  7. Use HTML5, CSS, JavaScript, libraries such as jQuery and development frameworks such as Zurb Foundation.
  8. Design usable forms for web pages and use the data.
  9. Interact with servers using basic Unix protocols and commands such as FTP.
  10. Explain the current status of cross browser and platform compatibility and the value of testing.
  11. Demonstrate testing and problem solving skills by using a variety of industry debugging tools.
  12. Explain the constantly changing culture of web development and strategies to update and extend personal web design skills.

Notes

Personal Standards

The work that you build in this class is only worthy of being part of your portfolio if the contents are all yours. It is okay to use logos and images you have developed in other classes. It is not a portfolio builder if you are using someone else’s art. It is dishonest if you don’t cite your sources in the deliverable.
